Instructions for Printing Single-Sided Cards

This page is designed for a single-sided print. To print these cards correctly, follow these steps in your browser's print dialog:

  1. First, ensure you have downloaded all the image files from your GitHub repository and that they are in the same folder as this HTML file.
  2. Open the print dialog by pressing Ctrl+P (or Cmd+P on Mac).
  3. In the print settings, make sure the paper size is set to A4.
  4. Set the **Margins** to None or Zero.
  5. Make sure "Print on both sides" or any duplex printing option is **turned off**.
  6. Click Print.

You can ignore the text on this page once you open the print dialog, as it will automatically hide for printing.
