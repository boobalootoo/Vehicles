Instructions for Printing Double-Sided Cards

To get a perfectly aligned, double-sided printout, follow these steps in your browser's print dialog:

  1. First, make sure you have downloaded all the image files from your GitHub repository and that they are in the same folder as this HTML file.
  2. Open the print dialog by pressing Ctrl+P (or Cmd+P on Mac).
  3. In the print settings, make sure the paper size is set to A4.
  4. Set the **Margins** to None or Zero.
  5. Look for an option for "Duplex" or "Print on both sides" and select it.
  6. Crucially, choose the option to "Flip on long edge". This ensures the back side of the card is correctly aligned.
  7. Click Print.

You can ignore the text on this page once you open the print dialog, as it will automatically hide for printing.
